## Ambiente Dev

Builde com Gradle

```shell
make build
```

Suba seu projeto com docker
```shell
make dk-up
```

Você pode ativar o Hotswap no IntelliJ iniciando um novo Spring Remote

Também pode debugar no IntelliJ iniciando uma chamada remota na porta 5005
